SPAship Deployer is up and running!

Deploy a new release or watch Logs using the available rest apis, listed below

Names of available pods, managed by light cd app

@Get: /pods/list

Watch last n lines of a managed pod

@Get: /pods/watch/<name-of-the-pod>?line=100

Delete all resources managed by light cd app

@Delete: deploy

Update an existing deployment(will also create related resources if not exists)

@Post: deploy

Body to deploy/update spaship-ui in a specific environment

{ "name":"spaship-ui", "repository":"spaship/spaship-ui", "namespace":"spaship", "docker_url":"quay.io/spaship/spaship-ui", "homepage":"https://quay.io/repository/spaship/spaship-ui", "updated_tags":[ "develop" ] }

the updated_tags specifies in which environment the resource is going to be deployed

CURL: curl --location --request POST 'http://light-cd-arbhatta-stage.apps.sandbox-m2.ll9k.p1.openshiftapps.com/deploy' \ --header 'Content-Type: application/json' \ --data-raw '{ "name": "spaship-ui", "repository": "spaship/spaship-ui", "namespace": "spaship", "docker_url": "quay.io/spaship/spaship-ui", "homepage": "https://quay.io/repository/spaship/spaship-ui", "updated_tags": [ "develop" ] }'
